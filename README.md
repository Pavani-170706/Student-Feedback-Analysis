## 📌 Overview

Student Feedback Analysis is a Data Mining project developed to analyze and understand student feedback data.

The project focuses on preprocessing feedback datasets, applying data mining techniques, and extracting meaningful insights from student responses to support better decision-making.


## ✨ Features

✔ Dataset preprocessing and cleaning

✔ Student feedback classification

✔ Clustering analysis

✔ Data visualization

✔ Pattern identification from feedback

✔ Report generation


## 📂 Dataset

The dataset contains student feedback records used for analysis.

Dataset files:

```
student_feedback_1000_records.csv

student_feedback_1000_records.arff
```
